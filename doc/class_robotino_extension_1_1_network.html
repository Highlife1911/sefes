Detailed Description

BasicNetwork adapter for communication.

Network adapter for communication between Robotinos.

This class implements a simple exchange of data over a custom NetworkId. The data gets supplied by a raw C-pointer. As an additional Parameter the size of the data is passed.

To allow communication there is also an set, that contains every valid NetworkId. The user can use it to keep track over valid communication targets.
