Low level movement function that needs to be called every 50 ms to keep moving. This function allows simultan movement in x and y direction while rotating.

Parameters

xspeed and direction on the x axis.
yspeed and direction on the y axis.
thetaspeed and direction of rotations.
Returns
EXIT_FAILURE or EXIT_SUCCESS.

The input is not checked and too big values are causing a missbehaving simulation.
