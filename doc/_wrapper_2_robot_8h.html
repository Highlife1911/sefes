EXPORT bool robotConnect ( void *  robot_handle)

Connects the robot and makes every other module usable. It is now possible to construct different modules with the robot handle.

Remarks
This method uses the ip the robot got constructed with.
Parameters
robot_handleHandle from the robot to connect.
Returns
EXIT_FAILURE or EXIT_SUCCESS.
EXPORT void* robotConstruct ( const char *  ip)

Constructs a robot from a ip.

Remarks
Does not actually connect to the robot. It is neccessary to call robotConnect after this command in order to use all modules.
Parameters
ipIP adress with port from the robot to connect with.
Returns
A new robot handle or Null on failure. Also might set an error Message.
