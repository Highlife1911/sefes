Detailed Description

GPS Positioning sensor.

This sensor retrieves the current cartesian position of the robot. The sensor is exact when used inside the simulator but not in the real world.

Remarks
There is a bug inside the current simulator version that swaps x and y values. For this reason the compile switch SWAP_X_Y exists that corrects this error.
